Native ads are ad assets presented using UI components that match your app’s look and feel.
In React Native, a loaded native ad is represented by `NativeAdContent`.

Native Ad loading in **React Native** is handled through `NativeAdLoader` from **react-native-cas**:

```ts
import {
  NativeAdLoader,
  NativeAdContent,
  AdChoicesPlacement,
  AdError,
  AdContentInfo,
} from 'react-native-cas';
```

## Add ad event listeners

Subscribe to loader events before requesting ads.

```tsx
import { useEffect } from 'react';
import { NativeAdLoader, NativeAdContent, AdError, AdContentInfo } from 'react-native-cas';

useEffect(() => {
  const unsubLoaded = NativeAdLoader.addAdLoadedEventListener((ad: NativeAdContent) => {
    console.log('Native Ad loaded', ad);
  });

  const unsubFailedToLoad = NativeAdLoader.addAdFailedToLoadEventListener((error: AdError) => {
    console.log('Native Ad failed to load', error);
  });

  const unsubImpression = NativeAdLoader.addAdImpressionEventListener((info: AdContentInfo) => {
    console.log('Native Ad impression', info);
  });

  const unsubClicked = NativeAdLoader.addAdClickedEventListener(() => {
    console.log('Native Ad clicked');
  });

  const unsubFailedToShow = NativeAdLoader.addAdFailedToShowEventListener((error: AdError) => {
    console.log('Native Ad failed to show', error);
  });

  return () => {
    unsubLoaded();
    unsubFailedToLoad();
    unsubImpression();
    unsubClicked();
    unsubFailedToShow();
  };
}, []);
```

<Info>
Read more about `AdContentInfo` in [Impression Level Data](ReactNative/Impression-Level-Data).
</Info>

## Load ad content

You can configure optional loader parameters:

- `NativeAdLoader.setStartVideoMuted(true | false)`
  Controls whether native video content starts muted.
  Recommended: `true` (better UX by default).

- `NativeAdLoader.setAdChoicesPlacement(...)`
  Preferred placement for the AdChoices icon inside your native ad view.
  Available values: `TOP_LEFT`, `TOP_RIGHT`, `BOTTOM_RIGHT`, `BOTTOM_LEFT`.

```ts
NativeAdLoader.setStartVideoMuted(true);
NativeAdLoader.setAdChoicesPlacement(AdChoicesPlacement.TOP_RIGHT);
```

Request one or more ads using `loadAds(maxNumberOfAds)`:

```ts
NativeAdLoader.loadAds(1);
```

A successfully loaded ad is delivered through `addAdLoadedEventListener`.

```tsx
import { useEffect, useState } from 'react';
import { NativeAdLoader, NativeAdContent } from 'react-native-cas';

const [nativeAd, setNativeAd] = useState<NativeAdContent | null>(null);

useEffect(() => {
  const unsubLoaded = NativeAdLoader.addAdLoadedEventListener((ad: NativeAdContent) => {
    // Optional safety check: do not show an expired ad.
    ad.isExpired().then(expired => {
      if (expired) {
        ad.destroy();
        return;
      }

      setNativeAd(prev => {
        prev?.destroy(); // optional: replace previous ad
        return ad;
      });
    });
  });

  NativeAdLoader.loadAds(1);

  return () => {
    unsubLoaded();
    setNativeAd(prev => {
      prev?.destroy();
      return null;
    });
  };
}, []);
```

## Load multiple ads (optional)

Request multiple native ads by passing a larger number to `loadAds`.
Each loaded ad will be delivered via `addAdLoadedEventListener` (potentially multiple times).

```ts
NativeAdLoader.loadAds(3);
```

One simple approach is to keep a small queue:

```tsx
import { useEffect, useState } from 'react';
import { NativeAdLoader, NativeAdContent } from 'react-native-cas';

const [queue, setQueue] = useState<NativeAdContent[]>([]);

useEffect(() => {
  const unsubLoaded = NativeAdLoader.addAdLoadedEventListener((ad: NativeAdContent) => {
    setQueue(prev => [...prev, ad]);
  });

  NativeAdLoader.loadAds(3);

  return () => {
    unsubLoaded();
    setQueue(prev => {
      prev.forEach(ad => ad.destroy());
      return [];
    });
  };
}, []);
```

## Display ad content

Native ad content is displayed using `NativeAdView`. There are two approaches:

- **Template view** — quickest way to display native ads (render `NativeAdView` without children).
  Continue with: [Native Ad templates](ReactNative/Native-Ads-Templates)

- **Manual Native Ad** — fully custom layout using asset components inside `NativeAdView`.
  Continue with: [Manual Native Ad](ReactNative/Native-Ad-Manual)

## Release ad resources

Always release a loaded native ad when it is no longer displayed.

```ts
nativeAd?.destroy();
```

You can also validate an ad before displaying it:

```ts
const expired = await nativeAd.isExpired();
if (expired) nativeAd.destroy();
```

<Warning>
- Do not display the same `NativeAdContent` in multiple views at the same time.  
- Always call `destroy()` when the ad is no longer used.
</Warning>
